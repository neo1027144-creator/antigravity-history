# antigravity-history

> 🔮 Export, recover, and analyze your Google Antigravity conversations — with full fidelity.

**⚠️ Coming soon — v0.1.0 under active development.**

The only tool that captures **AI thinking chains**, **code diffs**, **command outputs**,
and **timestamps** from your Antigravity sessions. Plus, recover conversations
lost after crashes or updates.

## Quick Start (coming soon)

```bash
pip install antigravity-history
aghistory export --output ./my-backup
```

## Features (planned)

- 🧠 Export AI's hidden thinking/reasoning process
- 📝 Full code diffs from every edit
- 💻 Command outputs and exit codes
- ⏰ Per-message timestamps
- 🔄 Recover conversations lost after crashes/updates
- 📁 Obsidian integration (frontmatter + wiki-links)
- 🖥️ Windows + macOS support
